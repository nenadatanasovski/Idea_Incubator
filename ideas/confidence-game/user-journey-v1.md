## Phase 3: Scene & Opportunity System

### Scene Variables (Randomly Generated)

| Variable          | Options                                                 | Impact                 |
| ----------------- | ------------------------------------------------------- | ---------------------- |
| Location          | Bus, train, cafe, party, etc.                           | Sets context/vibe      |
| Time pressure     | Relaxed / She's about to leave / Brief window           | Forces decision speed  |
| Opportunity type  | Eye contact made / Seated nearby / Bump into each other | Different openers work |
| Her initial state | Open/friendly / Distracted / In a hurry                 | Affects difficulty     |

### User Actions in Scene

1. **Wait and observe** â€” May miss the chance, but can read the room
2. **Approach immediately** â€” Bold, might score or backfire
3. **Create an opportunity** â€” (e.g., "accidentally" drop something nearby)
4. **React to random event** â€” Something happens that creates natural opener

### Timing Mechanics

- Some scenes have countdown (she's getting off next stop)
- Some are relaxed (library, she's studying)
- Miss the window = conversation doesn't happen, but you learn something

## Phase 4: Conversation System

### Girl Persona (AI Agent)

Randomly generated composite:

- **Personality type**: Shy / Confident / Playful / Intellectual / Mysterious / Warm
- **Current mood**: Happy / Stressed / Bored / Curious / Guarded
- **Interests**: 3-5 randomly assigned (music, sports, books, travel, etc.)
- **Communication style**: Direct / Subtle / Sarcastic / Earnest
- **Deal breakers**: 1-2 things that will end the conversation badly
- **Hidden likes**: Things that will make her light up if discovered

## Key Design Principles

1. **Never feel like a "lesson"** â€” It's a game that happens to teach
2. **Unpredictability is the feature** â€” Same approach won't always work
3. **Failure is entertaining** â€” Crashing and burning should be funny, not punishing
4. **Short sessions, high density** â€” Lots happens in 3 minutes
5. **The mystery** â€” Is she AI? Is she real? (Future feature, but plant the seed early)
