## State Management

### What Opens/Closes What

| Action                  | CycleIndicator | NodeInspector  | Resolution Panel |
| ----------------------- | -------------- | -------------- | ---------------- |
| Click node              | -              | Opens          | -                |
| Click "Resolve with AI" | Collapses      | Closes if open | Opens            |
| Close Resolution Panel  | -              | -              | Closes           |
| Click canvas (empty)    | Collapses      | Closes         | Closes           |
| Press Escape            | Collapses      | Closes         | Closes           |
| Click "Done" (success)  | -              | -              | Closes           |

### Panel Exclusivity

```
At any time, only ONE of these can be open:
- NodeInspector
- Resolution Panel

CycleIndicator (pill) can be expanded while panels are closed.
When Resolution Panel opens, CycleIndicator should collapse.
```
