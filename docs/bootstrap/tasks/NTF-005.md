## Summary

Implement email digests that summarize activity over a period. Users can choose daily or weekly digests instead of individual notifications.

---

## Requirements

1. **Digest Collection**:
   - Collect events over period
   - Group by type/idea
   - Generate summary

2. **Scheduling**:
   - Daily digest option
   - Weekly digest option
   - Configurable send time

3. **Template**:
   - Clean HTML email
   - Activity summary
   - Quick action links
