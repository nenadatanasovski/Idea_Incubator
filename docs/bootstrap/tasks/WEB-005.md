## Summary

Attention-grabbing modal for blocking questions that agents cannot proceed without.

---

## Requirements

1. **Modal Behavior**:
   - Auto-appears when blocking question arrives
   - Cannot be dismissed without answering
   - Stays on top of other content
   - Sound notification option

2. **Visual Design**:
   - High contrast, attention-grabbing
   - Clear "BLOCKING" indicator
   - Agent name and current task shown
   - Countdown if time-sensitive

3. **Quick Actions**:
   - Pre-filled options where possible
   - One-click common answers
   - Link to full detail view
   - Keyboard shortcuts
