## Summary

Create a multi-step task creation form with real-time validation feedback, test case builder, and acceptance criteria builder.

---

## Requirements

1. **Form Steps**:
   - Step 1: Basic info (title, description, category)
   - Step 2: Acceptance criteria builder
   - Step 3: Test case builder (API and UI)
   - Step 4: Review and submit

2. **Features**:
   - Real-time validation feedback
   - Category and priority selection
   - Auto-classification suggestions
   - Duplicate detection warning
