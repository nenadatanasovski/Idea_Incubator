## Summary

Create the ValidationResult component that displays validation results with blocking vs warning issues, fix suggestions, and auto-fix action buttons.

---

## Requirements

1. **Display Elements**:
   - Blocking issues (red, prevent execution)
   - Warning issues (yellow, allow execution)
   - Info issues (blue, informational)
   - Fix suggestions for each issue

2. **Actions**:
   - Auto-fix button (for autoFixable issues)
   - Dismiss warning button
   - View details link
