## Summary

Create 3 briefs of increasing complexity that will serve as test inputs for Spec Agent. These briefs will be used to generate reference specs.

---

## Requirements

1. **Simple Brief**: API call counter feature
   - Single database table
   - 2-3 API endpoints
   - No external dependencies

2. **Medium Brief**: User profiles feature
   - Multiple database tables with relationships
   - 5-7 API endpoints
   - File uploads or similar complexity

3. **Complex Brief**: Notification system
   - Multiple tables, queues, external services
   - WebSocket or real-time components
   - Multiple delivery channels

---

## Pass Criteria

**PASS** when ALL of the following are true:

| #   | Criterion                  | How to Verify                                                             |
