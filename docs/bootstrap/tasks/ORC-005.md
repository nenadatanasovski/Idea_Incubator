## Summary

Create an activity timeline showing all agent activities in chronological order. Part of the orchestration monitoring interface.

---

## Requirements

1. **Event Collection**:
   - Subscribe to all agent events
   - Buffer recent events
   - Support pagination

2. **Timeline Display**:
   - Chronological ordering
   - Event type icons
   - Time stamps

3. **Filtering**:
   - By agent
   - By event type
   - By time range
