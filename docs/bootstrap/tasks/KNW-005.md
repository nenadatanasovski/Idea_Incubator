## Summary

Implement logic to identify knowledge entries ready for promotion to CLAUDE.md. Promotion requires high confidence and human approval.

---

## Requirements

1. **Promotion Criteria**:
   - Confidence > 0.9
   - Usage count > 5
   - Prevention count > 3
   - Age > 7 days

2. **Proposal Generation**:
   - Draft CLAUDE.md section
   - Include rationale
   - Show evidence

3. **Approval Flow**:
   - Create proposal
   - Await human approval
   - Apply on approval
