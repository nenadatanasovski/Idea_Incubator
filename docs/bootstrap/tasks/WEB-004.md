## Summary

Full question detail page with context, options, and rich answer input.

---

## Requirements

1. **Question Context**:
   - Full question text
   - Context from agent (what prompted question)
   - Related files or code snippets
   - Links to relevant documentation

2. **Answer Options**:
   - Multiple choice if applicable
   - Text input for open questions
   - Option selection with description
   - Default option highlighted

3. **Submission**:
   - Submit answer
   - Skip with default
   - Request more context
   - Mark as "will answer later"
