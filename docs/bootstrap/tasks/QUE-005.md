## Summary

Implement a "answer all with defaults" feature that resolves all pending questions using their default answers. Useful for unblocking agents quickly.

## Requirements

1. **Bulk Action**:

   - Apply default answers
   - Resume blocked agents

2. **Filtering**:
   - Filter by priority
   - Filter by age
   - Exclude critical questions

3. **Confirmation**:
   - Show what will be answered
   - Require confirmation
   - Allow undo
