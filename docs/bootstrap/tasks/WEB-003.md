## Summary

Display pending questions with priority ordering, allowing users to answer inline or navigate to detail view.

---

## Requirements

1. **Question List**:
   - Priority-ordered display
   - Question type badge (BLOCKING/CLARIFYING/etc.)
   - Agent source indicator
   - Time since asked
   - Quick answer options if simple

2. **Filtering**:
   - Filter by agent
   - Filter by question type
   - Filter by answered/unanswered
   - Sort by priority or time

3. **Inline Actions**:
   - Quick answer for simple questions
   - Expand for details
   - Navigate to full detail view
   - Answer with defaults
