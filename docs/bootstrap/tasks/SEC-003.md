## Summary

Implement signed deep links for questions so users can answer directly from Telegram or email without logging in separately.

---

## Requirements

1. **Link Generation**:
   - Sign with secret key
   - Include question ID
   - Set expiry time

2. **Link Validation**:
   - Verify signature
   - Check expiry
   - Rate limit usage

3. **Direct Action**:
   - Show question on open
   - Allow answer without login
   - Invalidate after use
