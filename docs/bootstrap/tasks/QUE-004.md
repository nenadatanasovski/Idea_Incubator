## Summary

Implement question expiry and timeout handling. Questions that aren't answered within a time limit can auto-resolve with defaults or escalate.

---

## Requirements

1. **Expiry Configuration**:
   - Per-question-type timeouts
   - Grace period before expiry
   - Escalation thresholds

2. **Timeout Actions**:
   - Use default answer
   - Escalate to higher priority
   - Mark as abandoned

3. **Notification**:
   - Warn before expiry
   - Notify on expiry action
