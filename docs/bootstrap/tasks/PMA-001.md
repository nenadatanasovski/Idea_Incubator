## Summary

Use the Spec Agent to generate the technical specification for the PM (Project Manager) Agent. The PM Agent coordinates work between other agents and resolves conflicts.

---

## Context

### PM Agent Purpose

| Function                | Description               |
| ----------------------- | ------------------------- |
| **Work Distribution**   | Assign tasks to agents    |
| **Conflict Resolution** | Handle resource conflicts |
| **Priority Management** | Adjust task priorities    |
| **Deadlock Detection**  | Identify stuck workflows  |

---

## Requirements

1. **Prepare Brief**:
   - Document coordination requirements
   - Define conflict resolution rules
   - Specify escalation procedures

2. **Run Spec Agent**:
   - Generate spec from brief
   - Review for completeness
   - Human approval
