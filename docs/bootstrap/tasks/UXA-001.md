## Summary

Use the Spec Agent to generate the technical specification for the UX Agent. The UX Agent validates user experience through automated browser testing and accessibility checks.

---

## Context

### UX Agent Purpose

| Function | Description |
|----------|-------------|
| **Journey Testing** | Run multi-step user flows |
| **Accessibility** | Check WCAG compliance |
| **Screenshots** | Capture UI states |
| **Performance** | Measure load times |

---

## Requirements

1. **Prepare Brief**:
   - Document UX testing requirements
   - Define journey structure
   - Specify accessibility rules

2. **Run Spec Agent**:
   - Generate spec from brief
   - Review completeness
   - Human approval
