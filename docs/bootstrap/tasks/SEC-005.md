## Summary

Implement Telegram-based authentication flow. Users can link their Telegram account and authenticate via Telegram Login Widget.

---

## Requirements

1. **Telegram Login Widget**:
   - Add login button
   - Handle callback
   - Verify auth data

2. **Account Linking**:
   - Link Telegram to user
   - Store Telegram user ID
   - Enable quick auth

3. **Verification**:
   - Verify Telegram hash
   - Check data freshness
   - Secure session creation
