## Summary

Create the TaskTestResults component that displays test execution results with pass/fail status, failure details, screenshots, and re-run buttons.

---

## Requirements

1. **Display Elements**:
   - Pass/fail for each test case
   - Failure details and error messages
   - Screenshots for UI test failures
   - Execution duration

2. **Actions**:
   - Re-run single test
   - Re-run all tests
   - View screenshot full-size
   - Copy error to clipboard
