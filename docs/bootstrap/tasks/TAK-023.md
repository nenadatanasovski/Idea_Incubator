## Summary

Create the TaskCard component that displays task summary with status badge, validation indicator, and quick actions.

---

## Requirements

1. **Display Elements**:
   - Task title and description preview
   - Status badge with color coding
   - Validation status indicator
   - Category tag
   - Priority indicator

2. **Quick Actions**:
   - View details (expand)
   - Start/Pause execution
   - Validate
   - Edit
