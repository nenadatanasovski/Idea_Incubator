## Summary

Create the DependencyGraph component that visualizes task dependencies with blocked/unblocked status, interactive node selection, and critical path highlighting.

---

## Requirements

1. **Graph Features**:
   - Visualize task dependencies as directed graph
   - Show blocked/unblocked status per node
   - Interactive node selection
   - Critical path highlighting

2. **Interactions**:
   - Click node to view task details
   - Hover for task preview
   - Zoom and pan controls
   - Filter by status
