## Summary

Create real-time agent status display showing all agents, their states, and current activities.

---

## Requirements

1. **Agent Cards**:
   - Agent name and type
   - Current status (idle/working/blocked/error)
   - Current task description
   - Progress indicator if applicable

2. **Real-Time Updates**:
   - WebSocket subscription for agent events
   - Smooth state transitions
   - Pulse animation for active agents

3. **Visual States**:
   - Idle: Gray, muted
   - Working: Blue, animated pulse
   - Blocked: Yellow, attention grabbing
   - Error: Red, with error count
