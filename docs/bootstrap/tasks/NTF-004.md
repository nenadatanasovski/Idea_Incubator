## Summary

Implement browser push notifications as an alternative notification channel. Users can receive alerts directly in their browser even when not on the page.

---

## Requirements

1. **Service Worker**:
   - Register push service worker
   - Handle push events
   - Show notifications

2. **Permission Flow**:
   - Request notification permission
   - Store subscription
   - Handle denial gracefully

3. **Integration**:
   - Connect to notification dispatcher
   - Support priority levels
   - Include action buttons
