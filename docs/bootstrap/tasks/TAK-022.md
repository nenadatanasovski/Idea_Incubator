## Summary

Create the main Task Agent dashboard page showing task stats, recent tasks, and blocked tasks with real-time WebSocket updates.

---

## Requirements

1. **Dashboard Features**:
   - Task stats (total, by status, by category)
   - List recent tasks with quick actions
   - Show blocked tasks prominently
   - WebSocket connection for real-time updates

2. **Sections**:
   - Stats cards (Total, In Progress, Blocked, Completed)
   - Blocked tasks alert section
   - Recent tasks list
   - Task list selector
