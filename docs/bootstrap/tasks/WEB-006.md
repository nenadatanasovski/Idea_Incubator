## Summary

Create an activity timeline component that shows a chronological feed of system events. Updates in real-time via WebSocket.

---

## Requirements

1. **Event Display**:
   - Task status changes
   - Agent activities
   - User actions
   - System events

2. **Filtering**:
   - Filter by event type
   - Filter by agent
   - Filter by time range

3. **Real-time Updates**:
   - New events append
   - Smooth animations
   - Auto-scroll option
