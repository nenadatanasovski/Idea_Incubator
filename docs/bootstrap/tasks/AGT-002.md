## Summary

Hand-write spec.md and tasks.md for each of the 3 reference briefs. These become the gold standard that Spec Agent output is compared against.

---

## Requirements

1. **For Each Reference**:
   - `spec.md` following `templates/unified/build/spec.md` exactly
   - `tasks.md` with atomic, implementable tasks
   - Valid YAML frontmatter
   - Realistic gotchas

2. **Spec Quality**:
   - All template sections filled
   - Architecture decisions documented
   - Data model complete
   - API endpoints specified

3. **Task Quality**:
   - One file, one action per task
   - Correct phase ordering: database → types → queries → api → tests
   - Runnable validation commands
   - Complete code templates
