## Summary

Create a UI for managing notification preferences. Users can configure which channels receive which notification types.

---

## Requirements

1. **Channel Settings**:
   - Enable/disable Telegram
   - Enable/disable email
   - Enable/disable browser push

2. **Type Settings**:
   - Configure per notification type
   - Set urgency thresholds
   - Quiet hours

3. **UI Components**:
   - Settings page
   - Toggle controls
   - Save/cancel flow
